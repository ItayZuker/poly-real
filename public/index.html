What it is: The signing key for your wallet (EOA). The app derives the Signer address from it and uses it to place Polymarket orders. It is stored encrypted — never shown again after save.

Where to get it (any wallet that can export a private key):

  • MetaMask — Account details → Show private key
  • Rabby — Account / Address management → Export private key
  • Phantom / other EVM wallets — Settings → Export private key (if supported)
  • Dedicated bot wallet — generate a new key and fund only what you’re willing to risk

Any exportable EOA key works. Hardware wallets usually cannot export a key for server bots.
